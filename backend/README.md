# Agri-Allied Web Application

## Backend REST API

This project provides a REST API for managing crop information.

### How to Run Backend Locally

1. Clone the repository

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
npm start
```

4. Backend runs at

```
http://localhost:5000
```

### API Endpoints

- GET /api/crops
- GET /api/crops/:id
- POST /api/crops
- PUT /api/crops/:id
- DELETE /api/crops/:id
- GET /api/search?q=
